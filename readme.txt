22. Розробити клас "Функція", що представляє собою функцію виду y(x) = abs((abs(x - 2) + 1) / (x - 2)). Членом-даним класу є аргумент x, а одним із методів - обчислення значення функції для заданого аргумента. Обчислити значення функції на заданому проміжку із заданим кроком. Для значень, які не задовільняють області визначення функції, згенерувати відповідні виняткові ситуації і організувати їх обробку.
